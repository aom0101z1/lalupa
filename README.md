# La Lupa - Observatorio de Transparencia

Proyecto ciudadano de documentacion y verificacion de informacion publica en Colombia.

## Sobre este proyecto

Este sitio recopila y documenta informacion de fuentes periodisticas publicas y verificables sobre la gestion del gobierno colombiano.

## Fuentes

Toda la informacion proviene de medios de comunicacion reconocidos, verificadores de datos y fuentes oficiales. Cada caso incluye enlaces a las fuentes originales para verificacion independiente.

## Contribuir

Si deseas sugerir un caso documentado, utiliza el formulario de sugerencias en el sitio web.

## Licencia
